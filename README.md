# Phone Mouse Control

Control your computer's mouse using your phone as a touchpad.

## Download and run
```
curl -L https://github.com/tylerthecoder/phone-mouse-control/releases/download/main/phone-mouse -o phonemouse
chmod +x ./phonemouse
./phonemouse
```

## Development Setup

### Prerequisites
- Go 1.21 or later
- Node.js 18 or later
- qrencode
